# CISSP CAT 模擬試験システム

ISC² CISSP の本番試験形式（CAT：Computerized Adaptive Testing）に対応した模擬試験Webアプリです。

## 🌐 オンラインで使う（推奨）

**→ https://ak69143.github.io/cissp-cat-test/**

インストール不要。ブラウザからすぐに使えます。

---

## 特徴

- **CAT形式**：回答に応じて難易度が変化するアダプティブアルゴリズム実装
- **300問収録**：8ドメイン × 公式比率に沿った問題数、難易度3段階（Easy / Medium / Hard）
- **2つの試験モード**：模擬試験（CAT）とドメイン別練習
- **試験設定**：推定スコア表示・ヒント・解説を個別ON/OFF
- **本番試験モード**：スコア・ヒント・解説をすべて非表示にした厳密なシミュレーション
- **学習記録**：ドメイン別の正答率をブラウザに保存

## 試験仕様（ISC² 公式準拠 / 2024年4月更新版）

| 項目 | 内容 |
|---|---|
| 試験形式 | CAT（Computerized Adaptive Testing） |
| 問題数 | 100〜150問 |
| 制限時間 | 3時間 |
| 合格ライン | 1000点満点中700点 |
| 出題形式 | 4択（最も適切なものを選ぶ） |

## ローカルで動かす

`git clone` 後にローカルサーバーが必要です（`index.html` を直接ブラウザで開いても動きません）。

### VS Code + LiveServer（推奨）

1. [VS Code](https://code.visualstudio.com/) をインストール
2. LiveServer 拡張機能をインストール
3. `index.html` を右クリック → "Open with Live Server"

### Python

```bash
git clone https://github.com/ak69143/cissp-cat-test.git
cd cissp-cat-test
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

### Node.js

```bash
npx serve .
```

## 使い方

1. **模擬試験（CATシミュレーション）**
   - 「試験を開始する」をクリック
   - 試験設定（学習モード / 本番試験モード / カスタム）を選択
   - 100〜150問を解答後、スコアと合否を確認

2. **ドメイン別練習**
   - 「ドメインを選ぶ」から苦手なドメインを選択
   - 難易度フィルターで絞り込み可能
   - 回答後に解説・ヒントを活用
